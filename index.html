📊 Model Prediction Logic Explained

🔹 Five Simple Prediction Models

Naive Model (Naive Model) Simple
ŷ(t+h) = y(t)
The most basic forecasting method, assuming future values equal the last observed value. Suitable for stationary series without obvious trends and seasonality.
Prediction Logic Last Observed Value Repeated
Applicable Scenarios Short-term Forecasting, Random Walk
Seasonal Naive Model (Seasonal Naive) Simple
ŷ(t+h) = y(t+h-m×k)
Uses seasonal patterns for prediction, assuming future values at a given time equal the observed value from the same time in the previous cycle. For example, to predict next Monday's value, use this Monday's value.
Prediction Logic Same Period Historical Value
Seasonal Period (m) 7
Applicable Scenarios Strong Seasonal Data
Mean Model (Mean Model) Simple
ŷ(t+h) = (1/n) × Σy(i), i=1 to n
Uses the average of all historical data as the prediction. Assumes data fluctuates around a stable mean, ignoring trends and seasonality.
Prediction Logic Global Average
Applicable Scenarios Stationary Series, No Obvious Trend
Drift Model (Drift Model) Simple
ŷ(t+h) = y(t) + h × [(y(n) - y(1)) / (n-1)]
Adds a linear trend (drift term) to the Naive Model. Assumes data changes at a constant rate, determined by the slope between first and last observations.
Prediction Logic Last Value + Linear Trend
Drift Rate (Last-First)/(n-1)
Applicable Scenarios Data with Stable Trend
Dynamic Mean Model (Dynamic Mean / Rolling Mean) Simple
ŷ(t+h) = (1/k) × Σy(i), i=t-k+1 to t
Uses rolling average of the last k observations for prediction. Compared to global mean, better captures recent changes and is more sensitive to data variations.
Prediction Logic Rolling Window Average
Window Size (k) 5
Applicable Scenarios Locally Stationary, Short-term Fluctuations

🔸 Three Advanced Prediction Models

AR(p) AutoRegressive Model Advanced
ŷ(t) = c + φ₁×y(t-1) + φ₂×y(t-2) + ... + φₚ×y(t-p) + ε(t)
Automatically selects optimal order p based on PACF (Partial Autocorrelation Function). Computes PACF using Durbin-Levinson algorithm, selecting the last lag exceeding 95% confidence interval as p. Uses least squares to estimate model parameters.
Order Selection PACF Significance Test (α=0.05)
Max Search Order 30
Actual Order (p) Auto-selected
Parameter Estimation Multiple Linear Regression (OLS)
Applicable Scenarios Time Series with Strong Autocorrelation
STL Decomposition Extrapolation Advanced
y(t) = Trend(t) + Seasonal(t) + Residual(t)
Decomposes time series into trend, seasonal, and residual components. Trend extracted via moving average, seasonal component calculated from complete cycle averages. Components are extrapolated separately then combined: trend via linear regression, seasonal using corresponding cycle values.
Decomposition Method Moving Average + Cycle Average
Trend Extraction Centered Moving Average (window=m)
Seasonal Period (m) 7
Trend Extrapolation Linear Regression (Last 20 Points)
Applicable Scenarios Data with Clear Trend and Seasonality
Dynamic Rolling Mean (Dynamic Rolling Mean) Advanced
ŷ(t+1) = (1/k) × Σy(i), i=t-k+1 to t
Uses rolling mean with specified window size for one-step prediction. Similar to Dynamic Mean Model but participates in combined prediction as an independent advanced method. Window size can be adjusted based on data characteristics - larger windows smoother, smaller windows more responsive.
Prediction Logic Sliding Window Average
Window Size (k) 20
Applicable Scenarios Noisy Data Requiring Smoothing

🎯 Combined Prediction Strategy

Final Prediction = 0.7 × Advanced Model Mean + 0.3 × Simple Model Mean
Advanced Model Combination: AR(p) + STL Decomposition + Dynamic Rolling Mean (fixed weights 1/3 each)
Simple Model Combination: User-selected models (Naive/Seasonal Naive/Mean/Drift/Dynamic Mean), equal weight average
Weighted Fusion: 70% weight to advanced models, 30% to simple models, balancing complex pattern capture ability with simple model stability
