# 收盘价时间序列分析工具

这是一个基于Web的应用程序，用于分析和可视化Excel文件中的股票收盘价数据。

## 功能特点

- **Excel文件上传**：支持 .xlsx、.xls 和 .csv 格式
- **时间序列可视化**：使用Chart.js显示交互式图表
- **滚动缩放**：支持鼠标滚轮和拖拽缩放查看任意时间段
- **数据集分割**：自动将前80%数据标记为训练集，后20%标记为测试集
- **数据统计**：显示总数据量、训练集/测试集数量和日期范围

## 使用方法

### 1. 打开应用

直接在浏览器中打开 `index.html` 文件：

```
双击 index.html 文件
```

或者在浏览器地址栏输入：
```
file:///c:/Users/dannn/Desktop/final/index.html
```

### 2. 准备Excel文件

确保您的Excel文件包含以下两列：

| 日期 | 收盘价 |
|------|--------|
| 2024-01-01 | 100.50 |
| 2024-01-02 | 101.20 |
| ... | ... |

**格式要求：**
- 第一列：日期（支持格式：YYYY-MM-DD, YYYY/MM/DD）
- 第二列：收盘价（数值类型）
- 第一行可以是表头（如：日期、收盘价），也可以直接是数据

### 3. 上传文件

- 点击上传区域选择文件，或
- 直接将Excel文件拖拽到上传区域

### 4. 查看结果

上传成功后，您将看到：
- **数据统计信息**：总数据量、训练集/测试集数量等
- **交互式图表**：
  - 蓝色线条：训练集（前80%数据）
  - 红色线条：测试集（后20%数据）
- **交互功能**：
  - 鼠标悬停：查看具体日期的收盘价
  - 鼠标滚轮：缩放图表
  - 拖拽：平移图表
  - 重置缩放：恢复原始视图

## 技术栈

- **HTML5**：页面结构
- **CSS3**：样式设计
- **JavaScript**：逻辑处理
- **Chart.js**：图表渲染
- **SheetJS (xlsx)**：Excel文件解析

## 注意事项

1. 所有数据处理都在本地浏览器完成，不会上传到服务器
2. 建议使用现代浏览器（Chrome、Firefox、Edge、Safari）
3. Excel文件大小建议不超过10MB

## 故障排除

**问题：无法解析文件**
- 确保文件格式正确（.xlsx, .xls, .csv）
- 检查日期格式是否正确
- 确保收盘价列为数值类型

**问题：图表不显示**
- 检查浏览器是否支持JavaScript
- 尝试刷新页面重新上传
- 使用其他浏览器尝试
